Tableau Site Discovery

Tableau Site is an analytics platform for users to publish their dashboards and share insights.

Users have access to only contents that they are granted permissions to by content owners. This means that they would not be aware of potentially similar contents present on this site. This also means that they would not know who to look for if they wish to request access to a desired content.

By tapping on metadata drawn from the Tableau site and presenting them in an interactive UI, it enables creators to identify opportunities for collaborations and users to direct their access requests to content owners with certainty.

Data obtained from personal Tableau Online site.

BTS Lyrics

Ever felt like listening to songs with lyrics that convey an emotion you're feeling?

This project attempts to use zero-shot classification to assign probability of pre-defined emotions for 230 BTS song lyrics. Input a pre-defined emotion and songs with descending probability of this emotion are returned.

Pre-defined emotions used are: happy, confident, sad, fearful, angry, nervous.

To provide more flexibility, the input emotion need not be from the pre-defined list. Zero-shot classification can be used to classify this input emotion as a pre-defined emotion and the returned songs follow.

Analysis of Stack Overflow data

Project aims to answer the following questions:

  • User Activity (Questions) - when are users actively posting questions?
  • Word Cloud - what do users usually ask?
  • Question Similarity - what are similar questions with replies that users who posted a question can refer to?
  • User Activity (Answers) - when do users actively answer questions?
  • Reply Duration - how fast can a user expect answers to his/her question?
  • (WIP) Answers Sentiment - what is the general sentiment of the community of users who replied?
